安装指南
==============

本教程面向使用 Accelerate & NPU 昇腾的开发者，帮助完成昇腾环境下 Accelerate 的安装。

Accelerate 下载安装
--------------------

.. note::
    
    阅读本篇前，请确保已按照 :doc:`安装教程 <../ascend/quick_install>` 准备好昇腾环境！


安装 Accelerate 及依赖包
------------------------

.. code-block:: shell

  pip install torch==2.7.1 torch_npu==2.7.1 accelerate -i https://pypi.tuna.tsinghua.edu.cn/simple
